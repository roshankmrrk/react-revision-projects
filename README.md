#  React Revision — Mini Projects Collection

A structured, hands-on revision of **React.js fundamentals to advanced concepts**, built as a series of small, focused projects. Each folder tackles one core concept — from `useState` to Redux — so the progression itself shows a clear learning path from basics to state-management at scale.

>  Built while strengthening my MERN stack fundamentals as I prepare for full-time Frontend/MERN Developer roles.

##  Why this repo?

Instead of one big project, this repo documents **incremental mastery** of React concepts:

```
useState → useEffect → useRef/useCallback → Custom Hooks 
   → Routing → Context API → Redux Toolkit
```

Each project is intentionally small and self-contained, making the code easy to review — every folder is a runnable, isolated example of one concept done well.
